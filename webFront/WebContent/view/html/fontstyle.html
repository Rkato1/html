글꼴 관련 스타일

font-family 속성 : 텍스트의 글꼴을 지정할 때 사용한다.

선택자 { font-family : 글꼴이름; }의 형식으로 사용된다.

웹 폰트 제공 사이트 예시 : 구글 웹 폰트 사이트로
font style test(Modak)

font-size 속성 : 텍스트의 크기를 변경할 때 사용한다.

선택자 { font-size : 크기; }의 형식으로 사용된다.

  • 고정 크기 px 테스트
  • 가변크기 % 테스트

font-weight 속성 : 두꺼운 글씨를 표현할 때 사용한다.

선택자 { font-weight : normal | bold | bolder | lighter | 100 | 400 | 900; }의 형식으로 사용된다.

  • 굵은 글꼴
  • 원래보다 가늘게
  • 원래보다 굵게

font-variant 속성 : 작은 대문자로 변경할 때 사용한다.

선택자 { font-variant : 속성값; }의 형식으로 사용된다.

I Love You
I Love You

font-style 속성 : 텍스트의 기울임을 지정할 때 사용한다.

선택자 { font-style : normal | italic | oblique; }의 형식으로 사용된다.

italic
oblique

font 속성 : 글꼴 관련 스타일을 한번에 지정할 때 사용한다.

선택자 { font : font-style font-variant font-weight font-size/line-height font-famliy; }의 형식으로 사용된다.
